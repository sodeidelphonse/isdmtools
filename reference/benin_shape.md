# Shapefile of Benin Republic

A simplified sf object representing the national boundary of Benin

## Usage

``` r
benin_shape
```

## Format

An sf object with 1 row and 3 variables:

- name:

  Country name

- iso_a3:

  Country code

- geometry:

  The spatial geometry (multipolygon)

## Source

World Map Data from Natural Earth (<https://www.naturalearthdata.com/>)
